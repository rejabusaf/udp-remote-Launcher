# udp-remote-Launcher
An application/command launcher windows Front-end for iOS apps like Cmd Remote.

In the Cmd Remote iOS app, create buttons, Example:
  NAME: <any>
  ENDPOINT: Host(Windows), UDP Port: 55555
  PACKET CONTENTS: 01 
                    # 01 for Button 1 and so on
  BEHAVIOUR: Send 1x
